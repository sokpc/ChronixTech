🚀 Chronix Tech - Ecossistema Digital & Portfólio

Este projeto é um ecossistema digital desenvolvido para consolidar a autoridade técnica e apresentar soluções de engenharia de software de alto nível. Focado em um design moderno de Glassmorphism, o portal serve como portfólio profissional e hub de serviços para soluções em automação, IA e arquitetura de sistemas.

💻 Sobre o Projeto

O desenvolvimento foi pautado em criar uma interface fluida e responsiva, utilizando práticas de Clean Code e as tecnologias mais modernas de front-end estático. O projeto faz parte de um ciclo de desenvolvimento acadêmico (ADS - Univali) voltado à aplicação prática de soluções para o mercado corporativo.

🛠️ Tecnologias Utilizadas

Design: Figma (Glassmorphism, Dark Mode)
Front-end: HTML5 semântico e CSS3 avançado (Grid & Flexbox)
Fontes: Google Fonts (Plus Jakarta Sans)
Ícones: Font Awesome 6
Hospedagem: Vercel (CI/CD automático via GitHub)
Versionamento: Git

🎨 Características Visuais

Glassmorphism: Uso de efeitos translúcidos e backdrop-filter para uma estética premium.
Dark Mode Nativo: Interface focada em conforto visual e modernidade.
Responsividade Total: Layout adaptável para dispositivos móveis, tablets e desktops.
SEO Amigável: Estrutura de tags otimizada para motores de busca.
